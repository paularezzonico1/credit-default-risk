# Model Artifacts

Generated by **Phase 11** of `notebooks/Credit Default Risk Analysis.ipynb` and consumed by `app.py`.

| File | Contents |
|---|---|
| `cluster_scaler.joblib` | `StandardScaler` fit on the 5 clustering features |
| `kmeans.joblib` | `KMeans` (k=4) borrower-segmentation model |
| `xgb_model.joblib` | Tuned production XGBoost classifier (raw features) |
| `metadata.joblib` | dict: `features`, `cluster_features`, `threshold`, `test_auc` |

Regenerate with:
```bash
jupyter nbconvert --to notebook --execute --inplace "notebooks/Credit Default Risk Analysis.ipynb"
```
